Automatic Diamond

Do you want the computer to draw for you? In this project we will create a program that draws a diamond automatically. Click in this example to see the final result:

Click here

1) Create a new project, choose your own object so it can be the guide for your pen extension and place it where you want your drawing to start. If you want you can use the same one from the example.

2) First add a pen extension. On the lower left corner you'll see a button to add an extension, choose the pen one.

3) When you press the green flag make sure to start with a block that erases all the old drawings so that you start with a new drawing each time.

Hint: These new blocks will be found under the pen extension.

4) Use the pen down block to start making lines, then send your pen guide to the coordinates where you want your drawing to start.

5) In this project we will draw a diamond so we will send our guide to the following coordinates to make a line and start the drawing.

6) Continue making lines using coordinates to draw lines until you complete your drawing.

7) To make this drawing more fun change the pen color forever and you'll see a different diamond each time.

8) Now change the pen's size using the the random number block between -10 and 10 to make different sized diamonds.

Bonus: Try personalizing your drawing by adding wait blocks between the lines, using different coordinates to make a other shapes or anything else you can imagine. Be creative!

*** Important: Always remember to save your project and share it when you're done.
